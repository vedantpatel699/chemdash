Engineering Manual — Membrane Analyzer

1. Purpose & Scope

Real-time hollow-fiber membrane separator monitoring for H2 purification. Tracks permeance, recovery, purity, and mass balance integrity. Single-stage PSA/membrane simulation.

2. Key Metrics

Normalized flow (Q_norm) — H2 permeate flow as % of design baseline. Indicator of membrane aging and fouling.

Recovery — H2 permeate qty / H2 feed qty. Economic measure; typical 70-85% for single stage.

Stage cut (θ) — Total permeate / total feed. Controls pressure drop and selectivity. Usually 15-35%.

Permeate purity (y_H2) — Mol % H2 in permeate. Spec typically 75-99% depending on application.

Selectivity (α) — H2 preferential transport vs contaminants. Ratio of mole fractions × pressure ratio.

3. Input Variables (13)

Timestamp, Q_feed, y_feed_H2, P_feed, Q_perm, y_perm_H2, P_perm, Q_residue, y_residue_H2, P_residue, T (process temp C), and calculated ΔP_lm.

4. Calculations

Q_norm = (Q_H2_perm / Q_baseline) × 100

Recovery = (Q_H2_perm / Q_H2_feed) × 100

θ = (Q_perm / Q_feed) × 100

α = (y_H2 / (1 - y_H2)) × (P_res / P_perm)

MB_gap = |Q_feed - (Q_perm + Q_res)| / Q_feed × 100

5. Alert Tiers

Permeance drop: advisory 5%, alarm 10%. Purity: advisory if below 93 mol %. Recovery: advisory if below 75%. MB gap: advisory if above 3%.

6. Assumptions

Ideal gas; T held constant over stage. Permeance = baseline (586 Nm3/hr·bar) unless degraded. No fouling model, only performance deviation.
